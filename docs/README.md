# Architecture Diagrams

This directory contains `.drawio` architecture diagrams for ShieldAgent.

## Viewing the Diagrams

| Diagram | Description |
|---------|-------------|
| `architecture-level1-overview.drawio` | High-level system context |
| `architecture-level2-internal.drawio` | Internal component structure |
| `architecture-level3-dataflow.drawio` | Data flow through the scan pipeline |

### How to Open

- **VS Code**: Install the [Draw.io Integration](https://marketplace.visualstudio.com/items?itemName=hediet.vscode-drawio) extension
- **Browser**: Open [draw.io](https://app.diagrams.net/) and drag the file in
- **Desktop**: Download [draw.io Desktop](https://github.com/jgraph/drawio-desktop/releases)
